# Fake Job Detector

Fake Job Detector is a web tool that helps you check if a job or internship post might be a scam. 
It looks for warning signs like unrealistic salaries, urgent language, suspicious contacts, poor grammar, or offers that seem too good to be true.
It uses simple rules and a machine learning model to give a scam risk score.

# Features

1) Check Job/Internship Posts – Paste a job description and get a scam risk score.
2) Detect Red Flags – Finds common scam signs like Urgent language ("Apply now", "Immediate start"), Unrealistic salary, Poor grammar or excessive punctuation, etc.
3) Easy Web Interface – Built with Streamlit and it's link is [https://raddo-singh-coding-projects-fakejobinternshipdetector-3irvjb.streamlit.app/]


# How It Works

1. Rules Check – Looks for scam keywords and patterns.
2. Machine Learning – Predicts the probability of a post being a scam.
3. Scam Score – Combines all findings into a score (0–100%).


# Red Flags to Watch

* Urgent hiring or pressure language
* Unrealistic pay promises
* Contact via WhatsApp, Telegram, or DMs
* Bad grammar, too many punctuation marks, ALL CAPS
* Requests for money or deposits


# Disclaimer

This tool only gives a risk estimate. It is not 100% accurate.
